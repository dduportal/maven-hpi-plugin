hpi:record-core-location

Full name:

org.jenkins-ci.tools:maven-hpi-plugin:3.40:record-core-location

Description:

Keeps track of where core was built from, so plugins depending on a snapshot version of core can pick up changes without reload.

Attributes:

  • Requires a Maven project to be executed.
  • The goal is not marked as thread-safe and thus does not support parallel builds.
  • Binds by default to the lifecycle phase: package.

Parameters

(no parameters)
